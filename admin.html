زر تهيئة ورفع البيانات الافتراضية

سيقوم هذا الخيار بالاتصال بمشروع Appwrite الخاص بك ومحاولة إنشاء وثائق الوجهات والرحلات السياحية الافتراضية (16 رحلة و6 وجهات) داخل مجموعات البيانات الخاصة بك دفعة واحدة.

ملاحظة هامة: يجب إنشاء مجموعات البيانات (Collections) داخل لوحة تحكم Appwrite الخاصة بك أولاً بالمسميات والخصائص الموضحة في الدليل الجانبي قبل الضغط على هذا الزر.

دليل هيكل بيانات Appwrite (Schema Guide)

يرجى الدخول إلى لوحة تحكم Appwrite وإنشاء قاعدة بيانات بالمعرف المحدد، ثم إنشاء المجموعات (Collections) التالية وتعيين صلاحيات القراءة والكتابة (Permissions) لها للجميع (role:all أو role:guests) كالتالي:

الخصائص (Attributes):
  • id : Integer
  • name_ar : String
  • name_en : String
  • desc_ar : String (size 1000)
  • desc_en : String (size 1000)
  • category : String
  • is_featured : Boolean
  • sort_order : Integer
  • image : String (URL size 500)

الخصائص (Attributes):
  • id : Integer
  • title_ar : String
  • title_en : String
  • country_ar : String
  • country_en : String
  • price : Float
  • currency : String
  • duration : Integer
  • category : String
  • climate : String
  • travel_type : String (Array)
  • budget_tier : String
  • is_egyptian : Boolean
  • spots_total : Integer
  • spots_left : Integer
  • departure_dates : String (Array)
  • desc_ar : String (size 1000)
  • desc_en : String (size 1000)
  • highlights_ar : String (Array)
  • highlights_en : String (Array)
  • destination_id : Integer (Nullable)
  • image : String (URL size 500)

الخصائص (Attributes):
  • reference : String
  • trip_id : Integer
  • trip_title : String
  • name : String
  • email : String
  • phone : String
  • adults : Integer
  • children : Integer
  • travel_date : String
  • payment_method : String
  • total_price : Float
  • notes : String (size 1000)
  • status : String
  • created_at : String
